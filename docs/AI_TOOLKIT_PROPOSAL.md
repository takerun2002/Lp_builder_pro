## 🧠 なぜこの戦略か？

### 1. 特典にすべきもの（TRIZプロンプト等）

**理由**:
- 🎣 **フック効果**: 「これが無料でもらえるの？」で興味を引く
- 🔄 **エコシステム誘導**: 特典使用 → LP Builder Pro に価値を感じる → 本体利用
- 📢 **口コミ発生**: 「このプロンプトすごい！」→ LP Builder Pro の認知拡大
- 🎯 **ターゲットフィルタ**: AIツール使える人 = LP Builder Pro のターゲット

### 2. 本体に搭載すべきもの（YAML構築機能）

**理由**:
- 🔗 **コア体験との統合**: リサーチ結果 → ナレッジ化 → Magic Pen精度UP
- 💰 **継続利用促進**: 使えば使うほど「自分専用AI」が育つ → 離脱しない
- 🏰 **競合参入障壁**: 蓄積されたナレッジは移行しにくい = ロックイン
- 📈 **データ資産**: ユーザーのナレッジが溜まる = サービス価値向上

## 🛠️ YAML構築機能の詳細設計

### 機能概要

```
┌──────────────────────────────────────────────────────────────────┐
│                    ナレッジ構築エンジン                           │
│               「あなたの知識をAI最適形式に変換」                  │
└──────────────────────────────────────────────────────────────────┘

入力:
┌─────────────────────────────────────────────────────────────────┐
│ • テキストファイル（txt, md）                                    │
│ • PDF（教材、資料）                                              │
│ • URL（ブログ記事、セールスレター）                             │
│ • 動画URL（YouTube、Vimeo）→ 文字起こし                        │
│ • 音声ファイル → 文字起こし                                     │
│ • 直接入力（コピペ）                                            │
└─────────────────────────────────────────────────────────────────┘
        ↓
処理（AI分析）:
┌─────────────────────────────────────────────────────────────────┐
│ 1. コンテンツ抽出                                               │
│ 2. 構造化（セクション分け）                                     │
│ 3. 分類（悩み/ベネフィット/証拠/CTA/ストーリー）                │
│ 4. パワーワード/キーフレーズ抽出                                │
│ 5. 重複除去・最適化                                             │
│ 6. YAML形式に変換                                                │
└─────────────────────────────────────────────────────────────────┘
        ↓
出力:
┌─────────────────────────────────────────────────────────────────┐
│ • YAML形式ナレッジファイル                                       │
│ • Cursor Skills (.mdc)                                           │
│ • ClaudeCode Skills                                              │
│ • カスタムGPTs 形式                                              │
│ • LP Builder Pro 内蔵ナレッジ                                    │
└─────────────────────────────────────────────────────────────────┘
```
