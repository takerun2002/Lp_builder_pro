## 🆕 Meta広告ライブラリ（Facebook Ad Library）

### なぜ最重要なのか

```
競合の広告 = リサーチ結果の結晶
- 何百万円もかけてテスト済み
- 効果があるから出稿し続けている
- コピー・クリエイティブ・CTAが丸見え
```

### 取得できる情報

| 情報 | 活用方法 |
|------|----------|
| **広告コピー** | 刺さるヘッドライン、本文パターン |
| **CTA** | 効果的なボタン文言 |
| **クリエイティブ** | ビジュアルの傾向 |
| **配信プラットフォーム** | FB/IG/Messengerの使い分け |
| **アクティブ期間** | 長く配信 = 効果あり |
| **ページ情報** | 競合の全広告を一覧 |

### 検索URL構造

```
https://www.facebook.com/ads/library/?
  active_status=active       # active/inactive/all
  &ad_type=all              # all/political
  &country=JP               # 国コード
  &q=ダイエット             # 検索キーワード
  &search_type=keyword_unordered
  &media_type=all           # all/image/video
```

### 実装ファイル

```
src/lib/research/scrapers/meta-ads.ts
```

### 使用例

```typescript
import { MetaAdsResearch } from '@/lib/research/scrapers/meta-ads';

// キーワードで検索
const ads = await MetaAdsResearch.search({
  query: 'ダイエット サプリ',
  country: 'JP',
  activeOnly: true,
  limit: 50,
});

// 競合分析
const analysis = await MetaAdsResearch.analyzeCompetitors([
  '競合A',
  '競合B',
  '競合C',
]);

console.log(analysis.insights);
// {
//   commonHeadlinePatterns: ['数字を含む', '疑問形'],
//   commonCtaTexts: ['今すぐチェック', '詳しくはこちら'],
//   averageTextLength: 120,
//   platformDistribution: { facebook: 80, instagram: 45 },
// }
```

```

- 匿名だから本音
- 質問タイトル = 悩みの言語化
- ベストアンサー = 求められている解決策
```

| 情報 | 活用方法 |
|------|----------|
| **質問タイトル** | 悩みの言語化パターン |
| **質問本文** | 詳細な悩みの内容 |
| **閲覧数** | 悩みの深さ・共感度 |
| **回答数** | 関心の高さ |
| **ベストアンサー** | 求められている解決策 |

```

```

## 🔄 リサーチフロー

```
┌──────────────────────────────────────────────────────────────┐
│                    リサーチフロー                            │
└──────────────────────────────────────────────────────────────┘

Step 1: 市場調査
┌─────────────────────────────────────────────────────────────┐
│ Infotop → 売れ筋商品・価格帯・トレンド                       │
│ Amazon  → 関連書籍・レビュー                                │
└─────────────────────────────────────────────────────────────┘
                            ↓
Step 2: 競合分析
┌─────────────────────────────────────────────────────────────┐
│ Meta広告ライブラリ → 競合の広告・CTA・クリエイティブ         │
│ 競合LP分析        → 構成・コピー・オファー                  │
└─────────────────────────────────────────────────────────────┘
                            ↓
Step 3: 悩み収集
┌─────────────────────────────────────────────────────────────┐
│ Yahoo知恵袋 → 生の悩み・質問パターン                        │
│ Amazon     → レビューから悩み抽出                          │
│ YouTube    → コメントから悩み抽出                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
Step 4: キーワード収集
┌─────────────────────────────────────────────────────────────┐
│ Google検索    → サジェスト・関連検索                        │
│ Meta広告      → 広告で使われているキーワード                │
│ 競合LP        → パワーワード抽出                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
Step 5: コンセプト生成
┌─────────────────────────────────────────────────────────────┐
│ 収集した情報 + 内田式フレームワーク + TRIZ                   │
│           ↓                                                  │
│ 競合が思いつかない革新的コンセプト                          │
└─────────────────────────────────────────────────────────────┘
```
