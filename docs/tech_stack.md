# Technical Stack

## Frontend
- **Framework**: Next.js 14.x (App Router)
- **UI Library**: React 18.x
- **Styling**: Tailwind CSS 3.x
- **Components**: shadcn/ui
- **State Management**: Zustand 4.x
- **Canvas/Drawing**: Fabric.js 5.x (for Magic Pen)
- **Drag & Drop**: React DnD 16.x

## Backend (Local)
- **API**: Next.js API Routes
- **Database**: SQLite 3.x (via `better-sqlite3`)
- **Storage**: Local File System

## AI & Data Processing
- **Image/Edit**: Gemini 3 Pro Image Preview (Key for Magic Pen)
- **OCR/Analysis**: Gemini 2.5 Flash
- **Scraping**: Playwright 1.x
- **Export**: html2canvas, jsPDF, sharp

## Environment
- **Node.js**: 18.x+
- **OS**: Windows/macOS/Linux
