# Project Overview: LP Builder Pro

## Vision
**Webデザイナー・マーケターのクライアントワークを革新的に効率化するAI搭載LP制作ツール**
「マジックペンで"ここ"を指すだけ。言語化の壁を超えて、思った通りのLPを高速で作る」

## Core Value Proposition
- **Magic Pen**: Visually select areas to edit with AI (coordinate-based).
- **Price Destruction**: One-time purchase (29,800 JPY) vs Subscription.
- **Workflow Optimization**: From scraping -> editing -> comparison -> export.

## MVP Features (Priority: High)
1.  **Section-based Creation**: Edit sections individually.
2.  **Magic Pen**: The core differentiator.
3.  **LP Scraping**: Full page capture & analysis (Playwright + Gemini).
4.  **Color Dropper**: Extract colors from anywhere (EyeDropper API).
5.  **ABCD Comparison**: Compare 4 variations side-by-side.
6.  **Responsive Preview**: PC/Tablet/Mobile views.
7.  **Export**: PNG, PDF, HTML/CSS.

## Target Audience
- Web Designers, Marketers, Directors (Client Work).
