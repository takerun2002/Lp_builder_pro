### 7. 🟡 「キーワードバンク」自動構築

**発見**: Amazon書籍 + YouTube動画タイトルからの抽出

```python
# キーワード抽出パイプライン

async def build_keyword_bank(genre: str):
    keywords = []
    
    # 1. Amazon書籍タイトルから抽出
    books = await crawl4ai.scrape(
        f"amazon.co.jp/s?k={genre}&i=stripbooks"
    )
    for book in books:
        # レビュー数が多い = 売れてる = 刺さってる
        if book.review_count > 100:
            extracted = extract_power_words(book.title)
            keywords.extend(extracted)
    
    # 2. YouTube動画タイトルから抽出
    videos = await youtube_api.search(genre, limit=50)
    for video in videos:
        # チャンネル平均より伸びてる = タイトルが刺さってる
        if video.view_ratio > 1.5:
            extracted = extract_power_words(video.title)
            keywords.extend(extracted)
    
    # 3. Infotop商品名から抽出
    products = await scrape_infotop_ranking(genre)
    for product in products:
        extracted = extract_power_words(product.name)
        keywords.extend(extracted)
    
    # 4. Gemini でカテゴリ分類
    categorized = await gemini.categorize(keywords, categories=[
        "power_word",    # パワーワード
        "benefit",       # ベネフィット
        "pain",          # 悩み訴求
        "urgency",       # 緊急性
        "trust"          # 信頼性
    ])
    
    return categorized
```

**革命的ポイント**:
- 使うほど **キーワード辞典が成長**
- 「ダイエット系で刺さるパワーワードTOP50」が自動生成
- ジャンル横断でパターン発見
