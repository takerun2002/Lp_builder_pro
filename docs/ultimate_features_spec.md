## 📊 Google Sheets as Database

### シート構成

```
LP_Builder_Pro_Database (スプレッドシート)
├── 📄 projects           # プロジェクト一覧
├── 📄 research_results   # リサーチ結果
├── 📄 pain_points        # 悩みマトリックス
├── 📄 keywords           # キーワードバンク
├── 📄 competitors        # 競合分析
├── 📄 concepts           # コンセプト候補
├── 📄 concept_history    # コンセプト履歴（A/B結果付き）
└── 📄 settings           # 設定・マスタデータ
```

### 各シートの構造

#### projects シート
| project_id | name | genre | target | created_at | updated_at | status | owner |
|------------|------|-------|--------|------------|------------|--------|-------|
| uuid | LP案件A | beauty | 30代女性 | 2025-01-01 | 2025-01-05 | active | user@example.com |

#### pain_points シート
| id | project_id | content | source | depth | urgency | quadrant | original_text | created_at |
|----|------------|---------|--------|-------|---------|----------|---------------|------------|
| uuid | uuid | 彼女ができない | yahoo_chiebukuro | 5 | 5 | priority | 「もう3年彼女いない...」 | 2025-01-01 |

#### keywords シート
| id | project_id | word | source | source_title | performance_score | category | created_at |
|----|------------|------|--------|--------------|-------------------|----------|------------|
| uuid | uuid | 脈なし | youtube_video | 脈なし女子を... | 2.5 | power_word | 2025-01-01 |

#### concepts シート
| id | project_id | headline | target_pain | benefit | score_overall | is_selected | ab_result | created_at |
|----|------------|----------|-------------|---------|---------------|-------------|-----------|------------|
| uuid | uuid | 脈なしだろうが関係ない... | 女性のくどき方 | 好きな女性と付き合える | 88 | true | winner | 2025-01-01 |

### メリット
1. **無料で大容量** - 1000万セル/ファイル
2. **リアルタイム共同編集** - チーム全員でアクセス
3. **バージョン履歴** - 変更履歴が自動保存
4. **フィルタ/ピボット** - スプレッドシート本来の分析機能
5. **AppScript連携** - 自動化も可能
6. **API完備** - Google Sheets API v4

### 2. 🔌 Chrome拡張機能

```
機能:
- 競合LPを閲覧中に「リサーチに追加」ボタン
- ワンクリックでスクショ + URL + コンテンツを保存
- 知恵袋閲覧中に「悩みを追加」
- Amazon書籍ページで「タイトルを追加」

技術:
- Chrome Extension Manifest V3
- LP Builder Pro APIと連携
- Google Driveに直接保存
```

### 3. 📱 LINE Bot / Slack Bot

```
使い方:
1. 「リサーチ開始: ダイエット 30代女性」とメッセージ送信
2. バックグラウンドでリサーチ実行
3. 完了したらGoogle Docsレポートのリンクを通知

メリット:
- PCを開かなくてもリサーチ開始
- 移動中にアイデアを投げられる
- チームで共有しやすい
```

### 4. 🎤 音声入力リサーチ

```
「ダイエット系の商品を売りたい。ターゲットは30代の産後太りに悩む女性。

```

### 5. 🔄 A/Bテスト連携

```

- Twitter Ads連携

結果を自動収集してGoogle Sheetsに保存
→ 次回リサーチ時に「過去の勝ちパターン」として参照
```

### 6. 📚 ナレッジベース自動構築

```
リサーチ結果を蓄積することで:

→ 新規リサーチ時に自動参照

```
