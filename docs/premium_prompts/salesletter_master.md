# 🔥 セールスレター・マスタープロンプト

## LP Builder Pro 特典

**「デザイナー・LP制作者のための原稿作成AI」**

LP制作を依頼されたら、このプロンプトでクライアントにヒアリングするだけで
高品質なセールスレター原稿が完成します。

## 📋 使い方

### Cursor / ClaudeCode での使用方法

1. このファイルを `.cursor/rules/` または ClaudeCode の Skills に配置
2. 新規チャットで「@salesletter_master」で呼び出し
3. ステップに従って情報を入力

### 直接使用する場合

1. {} 内の赤字部分をクライアント情報で埋める
2. AIチャット（Claude/ChatGPT）にコピペ
3. 対話形式でセールスレターを完成

```
┌─────────────────────────────────────────────┐
│ 1. ヘッドコピー（ファーストビュー）         │
│    → キャッチコピー、メインビジュアル        │
├─────────────────────────────────────────────┤
│ 2. ストーリー・問題提起                     │
│    → 共感を得る導入、ターゲットの悩み       │
├─────────────────────────────────────────────┤
│ 3. 煽り立て or 共感                         │
│    → 放置した時の損失、緊急性               │

│ 4. 主張と証拠・商品の提示（解決策）         │
│    → 商品紹介、なぜ効果があるのか           │

│ 5. ブレット                                 │

├─────────────────────────────────────────────┤

│    → 社会的証明、第三者の評価               │
├─────────────────────────────────────────────┤
│ 7. オファー・限定性                         │
│    → 価格、特典、期間限定要素               │
├─────────────────────────────────────────────┤

├─────────────────────────────────────────────┤

│    → 最後の一押し、要約、CTA               │
```

# 📝 セールスレター作成プロンプト

## PHASE 1: 情報収集（クライアントヒアリング）

### Step 1: 基本情報入力

```
#役割
あなたはセールスレター専門のコピーライターです。
競合他社が溢れる中でも、お客様が即決で購入するコピーを書く専門家です。

#販売したい商品
{商品・サービスの詳細を入力}
- どんな悩みを持つ人が
- どんな手法・プロセスで
- どんな結果を得られるか
- サービスの形態（動画講座/コンサル/教材など）

#販売価格
{価格を入力。プランがある場合はプラン別に記載}
例：
- ライトプラン：29,800円
- スタンダード：59,800円
- プレミアム：129,800円

#商品名
{商品・サービスのタイトル}
```

### Step 2: ターゲット選定

```
#ターゲット選定の指示
上記の商品を、この価格で即決購入する「たった1人の理想の顧客」を
具体的に定義してください。以下の項目を全て埋めてください。

【基本プロフィール】
・お名前（仮名）：
・性別：
・年齢：
・職業：
・年収：
・家族構成：

【行動特性】
・休みの日の過ごし方：
・情報収集の方法：
・普段使うSNS：
・購入時の決済方法（分割回数含む）：

【心理特性】
・TOP3の悩み：
・この商品を買う決定的な理由：
・購入前に必ず聞く質問：
・購入後の理想の変化：

なぜこの人物を選定したか理由も説明してください。
```

```
#緊急性のある悩みTOP3
選定したターゲットにとって、今すぐこの価格を払ってでも
解決したい「緊急性のある悩み」をTOP3で教えてください。

選定基準：
- 今すぐ解決しないとマズイと感じている
- お金を払ってでも解決したい
- 商品の購入動機に直結する

1:
2:
3:
```

### Step 4: 悩みの深掘り

```
#悩みの深掘り
TOP3の悩みそれぞれについて、以下を深掘りしてください：

【悩み1について】
・いつからこの悩みを抱えているか：
・解決のために今まで試したこと・投資額：
・なぜ今すぐ解決する必要があるか：
・悩みの原因を本人はどう認識しているか：
・放置した場合の1年後の最悪シナリオ：

【悩み2について】
（同様に記入）

【悩み3について】
（同様に記入）
```

## PHASE 2: キャッチコピー作成

### Step 6: キーワード抽出

```
#キャッチコピー作成 STEP1

ここまでの情報を全て確認してください。

次に、ターゲットがSNS広告で見たら
「無意識でクリックせずにはいられない」キーワードを
30個リストアップしてください。

条件：
- ありきたりなキーワードはNG（競合も使っている）
- ターゲットの悩みをピンポイントで突く
- 「これは自分のことだ」と思わせる
- 喉から手が出るほど欲しいベネフィットを表現
```

### Step 7: キーワード選定

```
#キャッチコピー作成 STEP2

30個のキーワードから、この商品を即決購入する
ターゲットを最も引き寄せるキーワードを10個選んでください。

選定基準：
- 即決で購入する人を集客できるか
- 価格に見合う価値を感じさせるか
- 競合と差別化できるか
```

### Step 8: キャッチコピー案作成

```
#キャッチコピー作成 STEP3

選んだ10個のキーワードを使って、
3パターンのキャッチコピーを作成してください。

必須要素：
□ ターゲットが最も興味を惹かれるフレーズ
□ この価格を払ってでも欲しいベネフィット
□ 競合にはないユニークな印象
□ 読めば悩みが解消しそうな直感性

各キャッチコピーを100点満点で自己評価し、
100点でない場合は改善してください。
```

### Step 9: キャッチコピー最終調整

```
#キャッチコピー最適化

「{良いと思ったキーワードやフレーズを列挙}」

これらを組み合わせて、さらに3パターン作成してください。
100点満点で自己評価し、100点になるまで改善してください。

#採用するキャッチコピー
{最終決定したキャッチコピーをここに記載}
```

## PHASE 3: セールスレター本文作成

### Step 10: セールスレター執筆

```
#セールスレター作成指示

以下の10セクション構成でセールスレターを書いてください。

【前提ルール】
1. ターゲットが即決で購入するよう書く
2. 競合より優れたユニークな価値を約束する
3. 全体に証拠要素を散りばめる
   - 主張に対する理由
   - 実績提示
   - お客様の声
   - メカニズム説明
   - 具体的な数字
   - 保証の提示
4. 読者の本音に応える
   - 「自分のための内容だ」と感じさせる
   - 心や感情を揺さぶる表現
   - 小学生でもわかる言葉
   - 読むだけで問題が一部解消する価値

─────────────────────────────────────
【セクション1: ヘッドコピー】
─────────────────────────────────────
採用したキャッチコピーをそのまま使用。
サブコピーで補足説明を追加。

─────────────────────────────────────
【セクション2: ストーリー・問題提起】
─────────────────────────────────────
「あなたは〇〇のような悩みを抱えていませんか？」
ターゲットが「うん、それは悩んでる」と深く頷く悩みを提示。
3つの悩みをリストアップ。
「1つでも当てはまるなら最後まで読んでください」

─────────────────────────────────────
【セクション3: 煽り立て or 共感】
─────────────────────────────────────
悩みの原因を提示（当たり前の原因はNG）。
放置した場合の最悪シナリオを提示。
「絶対に避けたい」と思わせる恐怖心を適度に煽る。

─────────────────────────────────────
【セクション4: 主張と証拠・商品の提示】
─────────────────────────────────────
「もしあなたが〇〇の結果を手に入れたいなら...」
実績を提示（自分のサクセスストーリー or 箇条書き）。
商品を発表「それが、〇〇です」

─────────────────────────────────────
【セクション5: ブレット（商品内容）】
─────────────────────────────────────
商品内容を1つずつ、ベネフィット訴求で紹介。
「〇〇することで、△△が手に入ります」形式。
各コンテンツの価値が価格以上に感じられるよう。

─────────────────────────────────────
【セクション6: お客様の声・推薦者の声】
─────────────────────────────────────
クライアントの実績をBefore→After形式で紹介。
具体的な数字と感情的な変化を含める。

─────────────────────────────────────
【セクション7: オファー・限定性】
─────────────────────────────────────
価格提示「〇〇の結果にいくらの価値がありますか？」
通常価格 → 今回の特別価格の流れ。
なぜこの価格なのか理由を説明。
限定性（期間/人数/特典）を追加。

─────────────────────────────────────
【セクション8: 保証（リスクをなくす）】
─────────────────────────────────────
返金保証/サポート保証の内容。
「リスクは私たちが負います」メッセージ。

─────────────────────────────────────
【セクション9: Q&A（不安解消）】
─────────────────────────────────────
想定される質問と回答を5〜7個。
購入を迷う理由を全て潰す。

─────────────────────────────────────
【セクション10: 追伸（クロージング）】
─────────────────────────────────────
「今あなたには2つの道があります」
1つ目：自分で努力する道（時間とお金を浪費）
2つ目：この商品を手に入れる道（最短で結果）

オファー内容を簡潔にまとめ。
「今すぐ下のボタンをクリック」でCTA。
─────────────────────────────────────

テキストファイルとして出力してください。
```

## 💡 LP制作者向けTIPS

### クライアントへのヒアリングシート

以下の情報をクライアントから事前に収集しておくと、原稿作成がスムーズです：

```
【必須情報】
□ 商品・サービスの詳細
□ 販売価格（プラン別）
□ 競合との違い
□ 自分の実績
□ クライアントの実績（3件以上）
□ 特典内容
□ 保証内容

【あると良い情報】
□ 過去に反応が良かったコピー
□ 既存客の声（原文）
□ 競合のLP URL
□ NGワード・使いたくない表現
□ ブランドのトーン&マナー
```

### デザイナーへの指示出し

セールスレター完成後、以下の形式でデザイナーに渡すと効率的です：

```
【セクション1: ヘッドコピー】
テキスト：〇〇〇〇
デザイン指示：
- ファーストビュー、インパクト重視
- 背景画像：〇〇なイメージ
- 文字色：〇〇

【セクション2: ストーリー】
テキスト：〇〇〇〇
デザイン指示：
- 共感を生む柔らかいデザイン
- アイコンで悩みを可視化

...以下同様
```
