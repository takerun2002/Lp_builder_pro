## 🚀 使い方

### 方法1: Cursor / ClaudeCode で使う（推奨）

```
1. ファイルを配置
   - Cursor: `.cursor/rules/` に配置
   - ClaudeCode: Skills に追加

2. チャットで呼び出し
   「@salesletter_master を使ってセールスレターを作成して」

3. ステップに従って情報を入力

4. 完成したセールスレターをデザインに落とし込む
```

### 方法2: 直接AIチャットで使う

```
1. プロンプトファイルを開く

2. {} 内をクライアント情報で埋める

3. AIチャット（Claude / ChatGPT）にコピペ

4. 対話形式で完成させる
```

## 🔗 Cursor / ClaudeCode 連携

### Cursor Rules 設定例

```yaml
# .cursor/rules/salesletter.mdc

name: "セールスレター・マスター"
description: "LP原稿作成のための対話型プロンプト"
triggers:
  - "セールスレター"
  - "LP原稿"
  - "原稿作成"

content: |
  （salesletter_master.md の内容）
```

### ClaudeCode Skills 設定例

```
1. ClaudeCode の Settings を開く
2. Skills セクションに移動
3. 「Add Skill」で新規作成
4. プロンプト内容をペースト
5. トリガーワードを設定
```

## ⚠️ 注意事項

1. **AIの回答は必ず確認してください**
   - 実績や数字は勝手に作らせない
   - クライアント情報と照らし合わせる

2. **クライアントの確認を取ってから納品**
   - AI生成であることを伝えるかはケースバイケース
   - 最終確認は必ず人間が行う

3. **継続的なアップデート**
   - LP Builder Pro のアップデートで
     新しいプロンプト/ナレッジが追加されます
